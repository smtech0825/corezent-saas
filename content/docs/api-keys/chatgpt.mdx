ChatGPT API는 OpenAI가 제공하는 서비스로, **GPT-4o 등 고성능 모델**을 내 프로그램에 직접 연결할 수 있어요. 회원가입부터 GeniePost 연동까지 차근차근 따라해 보세요.

<Steps>

<Step>

### OpenAI 계정 만들기

1. [OpenAI 플랫폼](https://platform.openai.com)에 접속해요.
2. 오른쪽 상단의 **[Sign up]** 버튼을 클릭해요.
3. 이메일 주소를 입력하거나 Google / Microsoft 계정으로 간편 가입할 수 있어요.
4. 이메일로 가입했다면 인증 메일이 발송돼요. 메일함을 열어 **[Verify email address]** 버튼을 클릭해요.
5. 이름과 생년월일을 입력하고 가입을 완료해요.

<Callout type="info">
  이미 ChatGPT 계정이 있다면 같은 계정으로 로그인하면 돼요. ChatGPT와 API 플랫폼은 같은 계정을 공유해요.
</Callout>

</Step>

<Step>

### 결제 수단 등록 및 크레딧 충전

OpenAI API는 **선불 크레딧 방식**이에요. 먼저 카드를 등록하고 크레딧을 충전해야 API 키를 사용할 수 있어요.

1. 로그인 후 오른쪽 상단 계정 아이콘을 클릭하고 **[Billing]**을 선택해요.
2. **[Add payment method]**를 클릭해요.
3. 신용카드 또는 체크카드 정보를 입력하고 저장해요.
4. **[Add to credit balance]**를 클릭해서 원하는 금액만큼 충전해요. (최소 충전 금액: $5)

<Callout type="warn">
  카드 등록만 하고 크레딧을 충전하지 않으면 API 호출 시 오류가 발생해요. 반드시 크레딧을 먼저 충전하세요.
</Callout>

<Callout type="info">
  처음에는 $5~$10 정도만 충전해도 충분해요. 블로그 글 수십 편을 작성하고도 남아요.
</Callout>

</Step>

<Step>

### API 키 발급

1. 왼쪽 사이드바에서 [API keys] 메뉴를 클릭해요. 또는 [platform.openai.com/api-keys](https://platform.openai.com/api-keys)로 바로 이동할 수 있어요.
2. **[+ Create new secret key]** 버튼을 클릭해요.
3. 키 이름을 자유롭게 입력해요. (예: GeniePost, 자동포스팅 등)
4. **[Create secret key]** 버튼을 누르면 키가 생성돼요.
5. 화면에 표시된 API 키를 **지금 바로 복사**해서 메모장에 저장해요.

<Callout type="warn">
  이 창을 닫으면 API 키를 다시 볼 수 없어요! 반드시 복사해서 안전한 곳에 저장하세요.
</Callout>

<Callout type="info">
  API 키는 'sk-proj-'로 시작하는 긴 문자열이에요. 예: sk-proj-xxxxxxxxxxxxxxxxxxxxxxxx
</Callout>

</Step>

<Step>

### 월별 사용 한도 설정 (선택 권장)

예상치 못한 과금을 방지하기 위해 월별 사용 한도를 설정해 두는 것을 권장해요.

1. **[Billing]** 메뉴에서 **[Usage limits]**를 클릭해요.
2. 'Set a monthly budget'에 매월 최대로 쓸 금액을 입력해요. (예: $10)
3. **[Save]** 버튼을 눌러 저장해요.

<Callout type="info">
  한도 금액에 도달하면 API 호출이 자동으로 중단돼요. 처음에는 $10 정도로 설정해두면 안심이에요.
</Callout>

</Step>

<Step>

### 제품에 API 키 입력

지니워크(GenieWork)도 동일하게 프로그램 내 [설정]에서 API 키를 입력해요.

1. GeniePost를 실행하고 [설정] 탭으로 이동해요.
2. ChatGPT API 키 입력칸에 복사해둔 키를 붙여넣기(**Ctrl+V**)해요.
3. **[설정 저장]** 버튼을 눌러 저장해요.
4. 저장 후 실행 탭에서 AI 엔진을 'ChatGPT'로 선택하면 바로 사용할 수 있어요.

<Callout type="info">
  키를 직접 타이핑하면 오타가 날 수 있어요. 반드시 복사(Ctrl+C) 후 붙여넣기(Ctrl+V)로 입력하세요.
</Callout>

</Step>

</Steps>

2025년 기준이며, 가격은 변동될 수 있어요.

| 모델 | 입력 (1M 토큰) | 출력 (1M 토큰) | 블로그 글 1편 예상 |
| --- | --- | --- | --- |
| **GPT-4o** | $2.50 | $10.00 | 약 $0.05~0.10 |
| **GPT-4o mini** | $0.15 | $0.60 | 약 $0.003~0.005 |
| **GPT-3.5 Turbo** | $0.50 | $1.50 | 가장 저렴한 옵션 |

<Callout type="info">
  고품질 글이 필요하면 GPT-4o, 가성비를 원하면 GPT-4o mini를 추천해요. 최신 요금은 [OpenAI 요금 페이지](https://openai.com/api/pricing)에서 확인하세요.
</Callout>

</Callout>
