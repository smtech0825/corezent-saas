---

Claude는 Anthropic이 만든 AI 엔진으로, **자연스럽고 문학적인 문체**가 특징이에요. 브랜드 블로그나 감성적인 글쓰기에 특히 강해요. 회원가입부터 GeniePost 연동까지 차근차근 따라해 보세요.

<Steps>

<Step>

### Anthropic 계정 만들기

1. [Anthropic API 콘솔](https://console.anthropic.com)에 접속해요.
2. 화면 중앙의 **[Sign up]** 버튼을 클릭해요.
3. 이메일 주소를 입력하거나 **Google 계정**으로 간편 가입할 수 있어요.
4. 이메일로 가입했다면 인증 메일이 발송돼요. 메일함을 열어 **[Verify Email Address]** 버튼을 클릭해요.
5. 이름과 휴대폰 번호를 입력하고 SMS 인증을 완료하면 가입이 끝나요.

<Callout type="info">
  이미 Claude.ai 계정이 있어도 API 콘솔은 별도 가입이 필요할 수 있어요. console.anthropic.com에서 확인해 보세요.
</Callout>

</Step>

<Step>

### 결제 수단 등록 및 크레딧 충전

Claude API는 **선불 크레딧 방식**이에요. 카드를 등록하고 크레딧을 충전해야 API 키를 실제로 사용할 수 있어요.

1. 콘솔 왼쪽 사이드바에서 **[Billing]**을 클릭해요.
2. **[Add credit card]** 버튼을 클릭해서 신용카드 또는 체크카드 정보를 입력해요.
3. **[Buy credits]**를 클릭해서 원하는 금액만큼 충전해요. (최소 충전 금액: $5)

<Callout type="warn">
  카드만 등록하고 크레딧을 충전하지 않으면 API 호출 시 오류가 발생해요. 반드시 크레딧을 먼저 충전하세요.
</Callout>

<Callout type="info">
  처음에는 $5 정도만 충전해도 충분해요. 블로그 글 수십 편을 작성하고도 남아요.
</Callout>

</Step>

<Step>

### API 키 발급

1. 왼쪽 사이드바에서 **[API Keys]**를 클릭해요. 또는 [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys)로 바로 이동할 수 있어요.
2. **[Create Key]** 버튼을 클릭해요.
3. 키 이름을 자유롭게 입력해요. (예: GeniePost, 자동포스팅 등)
4. **[Create Key]** 버튼을 누르면 API 키가 생성돼요.
5. 화면에 표시된 API 키를 **지금 바로 복사**해서 메모장에 저장해요.

<Callout type="warn">
  이 창을 닫으면 API 키를 다시 볼 수 없어요! 반드시 복사해서 안전한 곳에 저장하세요.
</Callout>

<Callout type="info">
  API 키는 'sk-ant-'로 시작하는 긴 문자열이에요. 예: sk-ant-api03-xxxxxxxxxxxxxxxx
</Callout>

</Step>

<Step>

### 월별 사용 한도 설정 (선택 권장)

예상치 못한 과금을 방지하기 위해 월별 사용 한도를 설정해 두는 것을 권장해요.

1. 왼쪽 사이드바에서 **[Billing]** → **[Limits]**를 클릭해요.
2. 'Monthly spend limit'에 매월 최대로 쓸 금액을 입력해요. (예: $10)
3. **[Save]** 버튼을 눌러 저장해요.

<Callout type="info">
  한도 금액에 도달하면 API 호출이 자동으로 중단돼요. 처음에는 $10 정도로 설정해두면 안심이에요.
</Callout>

</Step>

<Step>

### 제품에 API 키 입력

지니워크(GenieWork)도 동일하게 프로그램 내 [설정]에서 API 키를 입력해요.

1. GeniePost를 실행하고 [설정] 탭으로 이동해요.
2. Claude API 키 입력칸에 복사해둔 키를 붙여넣기(**Ctrl+V**)해요.
3. **[설정 저장]** 버튼을 눌러 저장해요.
4. 저장 후 대시보드의 API 상태에서 Claude 항목이 **초록색 '정상'**으로 바뀌면 연동 완료예요.

<Callout type="info">
  키를 직접 타이핑하면 오타가 날 수 있어요. 반드시 복사(Ctrl+C) 후 붙여넣기(Ctrl+V)로 입력하세요.
</Callout>

</Step>

</Steps>
