DeepSeek은 중국 AI 기업이 만든 고성능 언어 모델로, **GPT-4 수준의 성능을 극히 저렴한 비용**으로 사용할 수 있어요. API 키 발급도 간단해서 5분이면 완료돼요.

<Steps>

<Step>

### DeepSeek 플랫폼 접속 및 회원가입

1. [DeepSeek 플랫폼](https://platform.deepseek.com)에 접속해요.
2. 오른쪽 상단의 **[Sign Up]** 버튼을 클릭해요.
3. 이메일 주소를 입력하고 **[Send Code]** 버튼을 눌러요.
4. 입력한 이메일로 인증 코드가 발송돼요. 메일함을 열어 6자리 코드를 확인하세요.
5. 인증 코드를 입력하고 비밀번호를 설정한 후 가입을 완료해요.

<Callout type="info">
  이미 계정이 있다면 [Log In]을 눌러 바로 로그인하면 돼요.
</Callout>

</Step>

<Step>

### API 키 발급

1. 로그인 후 왼쪽 사이드바에서 [API Keys] 메뉴를 클릭해요.
2. 오른쪽 상단의 **[Create new API key]** 버튼을 클릭해요.
3. 키 이름을 자유롭게 입력해요. (예: GeniePost, 자동포스팅 등)
4. **[Create API key]** 버튼을 누르면 키가 생성돼요.
5. 화면에 표시된 API 키를 **지금 바로 복사**해서 메모장에 저장해요.

<Callout type="warn">
  이 창을 닫으면 API 키를 다시 볼 수 없어요! 반드시 복사해서 안전한 곳에 저장하세요.
</Callout>

<Callout type="info">
  API 키는 'sk-'로 시작하는 긴 문자열이에요. 예: sk-xxxxxxxxxxxxxxxxxxxxxxxx
</Callout>

</Step>

<Step>

### 크레딧 충전 (유료 사용 시)

DeepSeek API는 소량의 무료 크레딧으로 시작할 수 있어요. 무료 크레딧 소진 후에는 아래 방법으로 충전하면 돼요.

1. 왼쪽 사이드바에서 [Top Up] 메뉴를 클릭해요.
2. 충전할 금액을 선택하고 결제 수단(신용카드 등)을 입력해요.
3. 결제가 완료되면 크레딧이 즉시 충전되고, 바로 사용 가능해요.

<Callout type="info">
  DeepSeek은 타 AI 대비 가격이 매우 저렴해요. $1(약 1,400원)로 블로그 글 수백 편을 작성할 수 있어요.
</Callout>

</Step>

<Step>

### 제품에 API 키 입력

지니워크(GenieWork)도 동일하게 프로그램 내 [설정]에서 API 키를 입력해요.

1. GeniePost를 실행하고 [설정] 탭으로 이동해요.
2. DeepSeek API 키 입력칸에 복사해둔 키를 붙여넣기(**Ctrl+V**)해요.
3. **[설정 저장]** 버튼을 눌러 저장해요.
4. 저장 후 실행 탭에서 AI 엔진을 'Deepseek'으로 선택하면 바로 사용할 수 있어요.

<Callout type="info">
  키를 직접 타이핑하면 오타가 날 수 있어요. 반드시 복사(Ctrl+C) 후 붙여넣기(Ctrl+V)로 입력하세요.
</Callout>

</Step>

</Steps>
