title: Gemini API 키 발급 방법
description: 구글 계정만 있으면 5분 안에 발급받는 Gemini API 키. 무료 티어가 제공되어 처음 시작하는 분께 가장 추천하는 엔진이에요.
---

Gemini는 구글이 만든 AI 엔진으로, GeniePost의 **기본(권장) 엔진**이에요. 무료 티어가 제공되기 때문에 **처음 시작하는 분께 가장 추천하는 API**예요. 구글 계정만 있으면 5분 안에 발급받을 수 있어요.

<Steps>

<Step>

### Google AI Studio 접속

1. [Google AI Studio](https://aistudio.google.com)에 접속해요.
2. 평소 사용하던 **구글(Google) 계정**으로 로그인해요. 별도 가입 없이 구글 계정만 있으면 돼요.
3. 처음 접속 시 서비스 이용약관 동의 화면이 나타나면 내용을 확인하고 **[Continue]**를 클릭해요.

<Callout type="info">
  구글 계정이 없다면 [accounts.google.com/signup](https://accounts.google.com/signup)에서 무료로 만들 수 있어요.
</Callout>

</Step>

<Step>

### API 키 발급

1. Google AI Studio 화면 왼쪽 사이드바에서 **[Get API key]** 버튼을 클릭해요.
2. **[Create API key]** 버튼을 클릭해요.
3. 'Search for a project' 창이 나타나면 기존 구글 클라우드 프로젝트를 선택하거나, **[Create API key in new project]**를 눌러 새 프로젝트를 자동으로 생성해요. (처음 발급하는 경우 새 프로젝트 생성을 선택하면 돼요.)
4. 잠시 후 API 키가 생성돼요. **[Copy]** 버튼을 눌러 키를 복사하고 메모장에 저장해요.

<Callout type="warn">
  API 키는 절대 다른 사람에게 공유하거나 인터넷에 올리지 마세요. 키가 노출되면 무단 사용으로 요금이 발생할 수 있어요.
</Callout>

<Callout type="info">
  API 키는 'AIza'로 시작하는 긴 문자열이에요. 예: AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXX
</Callout>

</Step>

<Step>

### 무료 티어 사용 안내

Gemini API는 **무료로 사용할 수 있는 한도(Free Tier)**가 있어요. 처음 시작하는 분은 별도 결제 없이 바로 사용할 수 있어요.

- **Gemini 2.0 Flash (무료 티어):** 분당 15회 요청, 하루 1,500회 요청까지 무료
- **Gemini 1.5 Flash (무료 티어):** 분당 15회 요청, 하루 1,500회 요청까지 무료

<Callout type="info">
  하루 1,500회면 블로그 글을 매일 수십 편씩 써도 남아요. 처음에는 무료 티어만으로 충분히 테스트할 수 있어요.
</Callout>

</Step>

<Step>

### 유료 전환 방법 (무료 한도 초과 시)

무료 한도를 초과하거나 더 빠른 속도가 필요하다면 구글 클라우드 결제를 활성화하면 돼요.

1. [Google Cloud Console](https://console.cloud.google.com)에 로그인해요.
2. 사이드바에서 **[결제(Billing)]**를 선택하고 결제 계정을 만들어요.
3. 신용카드 또는 체크카드 정보를 입력하고 저장해요.
4. 결제 활성화 후에는 무료 한도가 없어지고 사용량만큼 과금돼요.

- **Gemini 2.0 Flash (유료):** 입력 $0.10 / 출력 $0.40 (1M 토큰 기준) — 블로그 글 1편 약 $0.001 이하

<Callout type="info">
  유료 전환 후에도 요금은 매우 저렴해요. 월 $1~2 이하로 수백 편의 글을 작성할 수 있어요.
</Callout>

</Step>

<Step>

### 제품에 API 키 입력

지니워크(GenieWork)도 동일하게 프로그램 내 [설정]에서 API 키를 입력해요.

1. GeniePost를 실행하고 [설정] 탭으로 이동해요.
2. Gemini API 키 입력칸에 복사해둔 키를 붙여넣기(**Ctrl+V**)해요.
3. **[설정 저장]** 버튼을 눌러 저장해요.
4. 저장 후 실행 탭에서 AI 엔진이 'Gemini'로 선택되어 있으면 바로 사용할 수 있어요.

<Callout type="info">
  Gemini는 GeniePost의 기본 엔진이에요. 다른 API 키 없이 Gemini 키 하나만으로 바로 자동 포스팅을 시작할 수 있어요.
</Callout>

</Step>

</Steps>

## Gemini 모델별 요금 요약 (참고)

2025년 기준이며, 가격은 변동될 수 있어요.

| 모델 | 입력 (1M 토큰) | 출력 (1M 토큰) | 특징 |
| --- | --- | --- | --- |
| **Gemini 2.5 Flash** | $0.15 | $0.60 | 속도와 품질의 균형이 뛰어나 가장 추천 |
| **Gemini 2.0 Flash** | $0.10 | $0.40 | 가성비 최고, 무료 티어 제공 |
| **Gemini 1.5 Pro** | $1.25 | $5.00 | 최고 품질, 긴 문서 처리에 유리 |

<Callout type="info">
  최신 요금 및 모델 정보는 [Google AI Studio 요금 페이지](https://ai.google.dev/pricing)에서 확인하세요.
</Callout>

</Callout>
