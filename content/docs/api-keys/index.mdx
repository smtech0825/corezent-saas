지니워크·지니포스트의 AI 자동 작성 기능은 여러분이 발급한 **AI API 키**로 동작해요. API 키는 AI 서비스(Gemini·ChatGPT·Claude·DeepSeek)에 직접 요청을 보내기 위한 개인 인증키예요. 아래에서 사용하려는 엔진을 골라 발급 방법을 그대로 따라 하시면 돼요.

<Callout type="info">
  처음 시작하는 분은 **무료 티어가 있는 Gemini**부터 발급받는 것을 추천해요. 결제 없이 바로 자동 작성을 테스트할 수 있어요.
</Callout>

## 4종 한눈 비교

| 엔진 | 무료 티어 | 최소 충전액 | 키 접두사 | 특징 |
| --- | --- | --- | --- | --- |
| **Gemini** | 있음 (하루 1,500회) | 무료 시작 (별도 충전 불필요) | `AIza` | 기본·권장 엔진, 처음 시작에 가장 추천 |
| **ChatGPT** | 없음 (선불 크레딧) | $5 | `sk-proj-` | GPT-4o 등 고성능 모델 |
| **Claude** | 없음 (선불 크레딧) | $5 | `sk-ant-` | 자연스럽고 문학적인 문체 |
| **DeepSeek** | 소량 무료 크레딧 제공 | 선택 충전(별도 명시 없음) | `sk-` | GPT-4 수준 성능을 극히 저렴하게 |
