1. 판매 제품 목록
제품명 : 제품은 관리자 페이지에서 등록을 할꺼야.
카테고리: 데스크톱/웹/크롬 익스텐션
판매 방식: 구독,  단일 구매 둘 다
구독이라면: 월간, 연간 둘 다
번들(묶음 상품): 관리자 페이지에서 등록

2. 결제 시스템
Lemon Squeezy만 사용
이미 Lemon Squeezy 계정은 승인 대기중이야.

3. 라이선스 방식
구매 후 어떤 방식으로 소프트웨어를 사용하게 되는지
시리얼 키 발급 → 소프트웨어에서 입력해서 활성화
다운로드 링크 제공
라이선스 제한: 기기 수, 사용기간

4. My Page 기능 범위
구매 내역 조회
라이선스 키 확인
내 정보 수정 (이름, 이메일, 비밀번호)
이메일 인증

5. 관리자(Admin) 페이지 필요
주문 현황, 사용자 목록, 라이선스 관리 등 백오피스 필요

6. Manual 포털 방식
로그인을 한 상태에서만 링크가 활성화
Manual 메뉴 클릭 시 메뉴얼 웹사이트로 이동

7. 백엔드 방향
Next.js API Routes로 해결

DB는 Supabase 사용
JSONB 타입 (Changelog용):업데이트 내용을 텍스트가 아닌 JSON 구조로 저장
RLS (Row Level Security):내 라이선스 정보는 나만 볼 수 있다"는 보안 로직을 백엔드 코드 대신 DB 자체에서 처리
