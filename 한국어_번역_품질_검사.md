### 핵심 우선순위 (실제 사용자 노출 — 즉시 수정 권장)

- **#2 `Download` 버튼** — 플랫폼이 2개 이상인 구독에서 영어 버튼 노출 (명백한 누락)
- **#3·#4 ActivateClient 영문 상태값** — `라이선스 expired` 식으로 영어 그대로 표시
- **#5 contact API 에러** — 문의 폼에서 영어 토스트 그대로 노출
- **#1 웰컴 이메일 제목** — 신규 가입자가 받는 메일 제목만 영어

### 참고 사항

- **조사/어미 오류 · 번역 과잉: 0건** — 문법은 전반적으로 정확하고, 외래어("대시보드"·"다운로드"·"라이선스" 등)는 적절히 유지되어 과잉 번역 없음.
- **존댓말 혼용: 0건** — 사이트 전체가 `합니다체` + `~하세요`(명령) + `~해 주세요`(요청)로 일관됨.
- 가장 두드러진 패턴은 **영어 'you'를 '당신'으로 직역**(#10~13)과 **영문 fallback 값 잔존**(#2,3,4,17,18)임.
